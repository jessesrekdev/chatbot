## Quick Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + N` | **New Chat** | Creates a new conversation instantly |
| `Ctrl + K` | **Search** | Opens conversation search (works anywhere) |

| `Ctrl + /` | **Help** | Shows this shortcuts reference |

## Theme & Display

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl + Shift + D` | **Toggle Theme** | Switches between light and dark mode |

## Chat Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Enter` | **Send Message** | Sends your message (when input is focused) |
| `Shift + Enter` | **New Line** | Adds a line break in your message |
| `Ctrl + Shift + C` | **Clear Chat** | Clears current conversation messages |

## Window Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Escape` | **Close** | Closes any open modal, dialog, or sidebar |

## üí° Pro Tips

### Quick Search
Press `Ctrl + K` from anywhere to instantly access the conversation search. On mobile devices, it will automatically open the sidebar first.

### Fast Theme Toggle
Use `Ctrl + Shift + D` to quickly switch themes without opening settings. Perfect for adjusting to different lighting conditions.

### Efficient Messaging
- Use `Enter` to send messages quickly
- Hold `Shift` while pressing `Enter` to add multiple lines
- The input bar will show hints when focused

### Navigation Shortcuts
All navigation shortcuts work when you're **not** typing in the input field (except `Ctrl + K` which works everywhere).

### Visual Feedback
When you use a shortcut, you'll see a brief notification confirming the action with the shortcut key combination displayed.
